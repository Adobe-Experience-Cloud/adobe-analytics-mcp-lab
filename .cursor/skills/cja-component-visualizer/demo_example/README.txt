Demo example (static preview only)
====================================

This folder is a frozen fake-dataset example for preview — not your org's data view.
For real builds, see SKILL.md in the parent folder.

Open component_network_demo_example.html in a browser (same folder as visualization_data_demo_example.js).
No MCP or Python required to view.

Regenerate (from the skill folder, cja-component-visualizer):
  python demo_example/build_demo_example.py

Or cd into this folder and run:
  python build_demo_example.py
